Hotel San Carlos POS

Sistema POS y administrativo hotelero desarrollado como aplicación desktop multiplataforma utilizando React, Vite y Tauri.

Descripción:

Hotel San Carlos POS es un sistema administrativo orientado a la gestión hotelera, diseñado para optimizar procesos internos como reservaciones, control de habitaciones, entradas/salidas y administración general.

La aplicación fue desarrollada utilizando tecnologías modernas de frontend y escritorio híbrido, priorizando modularidad, escalabilidad y experiencia de usuario.

Funcionalidades Principales:
Autenticación de usuarios
Sistema de login
Rutas privadas protegidas
Control de acceso
Gestión hotelera:
Administración de habitaciones
Gestión de reservaciones
Control de entradas y salidas
Dashboard administrativo

Instalación y Ejecución:

Clonar repositorio
git clone https://github.com/edgarbv32/hotel-san-carlos-pos.git

Instalar dependencias
npm install

Ejecutar proyecto
npm run dev

Ejecutar aplicación Tauri
npm run tauri dev
